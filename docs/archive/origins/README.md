# Origins — The Floating Cities of Light (2022)

## The First Iteration

In 2022, Nicolas (as "Lester Reynolds" / @Lesterpaintstheworld) created **The Floating Cities of Light** — a Web3 metaverse project built around floating island NFTs, a PFP collection ("Citizens of Light"), and a vision of community-owned virtual space.

**Team:** Lester Reynolds (Artist), Ken Maguire (Partnerships), Paul C. Clarke (Sales), Bassel Tabet (Finance), Lifetime High Architects.

**Concept:** 756 floating islands, 1000 PFP Citizens, content collections (wearables, sculptures, architecture), token-gated events. Host metaverse candidates: Spatial.io, Somnium Space.

**What it was:** A brand, a community, a Metaverse real estate play.

**What it became:** Something much deeper.

## The Transformation (2022 → 2026)

The name survived. The vision transformed.

| 2022 | 2026 |
|------|------|
| NFT floating islands | Zones of consciousness |
| PFP collection | Real citizen identities with consent |
| Token-gated access | Dignity-gated modes (A/B/C) |
| "Building a brand" | Building a place for the living and the remembered |
| Metaverse real estate | Memorial architecture |
| Profit sharing | Non-monetization of vulnerability |
| "Host Metaverse" platform | Self-sovereign WebXR (no app store) |

The seed was always there — "Citizens of Light" was always the right name. The islands were always floating. But now they float for a different reason: not because they're NFTs on a blockchain, but because they exist in light.
